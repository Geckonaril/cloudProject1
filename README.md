# Cloud Project 1 — Kitap Kütüphanesi

Bu proje Flask (Python) backend ve React frontend kullanarak
geliştirilmiş bir RESTful web uygulamasıdır.
AWS EC2, RDS ve S3 üzerinde barındırılmaktadır.

## Teknolojiler
- Backend: Python / Flask
- Frontend: React
- Veritabanı: PostgreSQL (AWS RDS)
- Hosting: AWS EC2 + S3 + CloudFront
